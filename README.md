# HR Attrition Analysis & Dashboard

This project analyzes employee attrition using the IBM HR dataset and visualizes key trends in a Streamlit dashboard.

## Features
- Clean Power BI–style dashboard
- Attrition distribution
- Job role insights
- Monthly income trends
- Correlation heatmap
- Boxplots and comparison charts

## Folder Structure

HR-Attrition-Project/
│── app/app.py
│── data/WA_Fn-UseC_-HR-Employee-Attrition.csv
│── models/model.pkl
│── notebooks/IBM_Attribution.ipynb
│── requirements.txt
│── .gitignore

## Run the Dashboard
Install requirements:
pip install -r requirements.txt

Run app:
streamlit run app/app.py
